SIRACAS adalah aplikasi web penjualan produk cacing dan olahan organik berbasis Laravel. Aplikasi ini mendukung alur customer untuk melihat produk, mengelola keranjang, checkout, pembayaran, riwayat transaksi, review, dan klaim garansi. Admin dapat mengelola produk, customer, transaksi, notifikasi, ulasan, garansi, serta laporan penjualan.

## Identitas

- Kelompok: A5
- Mata Kuliah: PPL Agroindustri

## Fitur Utama

- Autentikasi untuk customer dan admin.
- Katalog produk dengan detail, ulasan, dan rating.
- Keranjang belanja, checkout, beli langsung, dan pembayaran melalui Midtrans.
- Pengelolaan transaksi, ongkos kirim, refund, status pesanan, dan garansi.
- Notifikasi untuk customer dan admin, termasuk kampanye notifikasi.
- Dashboard admin, laporan penjualan, dan export data ke Excel.

## Teknologi

- Laravel 13
- PHP 8.3+
- Tailwind CSS 4
- Vite 8
- Alpine.js
- Pest
- Maatwebsite Excel
- Midtrans Snap
